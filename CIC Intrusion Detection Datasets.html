Description

It is a widely respected, standard academic dataset for network security. It contains benign network traffic and a wide variety of common cyberattacks (e.g., Brute Force, DoS, Botnet), with all traffic fully labeled. It is perfect for those interested in "Cybersecurity."

Dataset and Metadata

https://www.unb.ca/cic/datasets/ids-2017.html

Reference

Iman Sharafaldin, Arash Habibi Lashkari, and Ali A. Ghorbani, Toward Generating a New Intrusion Detection Dataset and Intrusion Traffic Characterization, 4th International Conference on Information Systems Security and Privacy (ICISSP), Portugal, January 2018.
